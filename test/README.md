# Test Fixtures

This directoy collects various
test-cases, test-suites, test-scripts, and test fixtures.

## Running Tests

If you've established the development environment
as described in the top level [README](../README.md#how-to-run),
your shell will already have a commend alias `t`
for aggregating and running tests.
The `t` alias runs the command:

    $ deno task test

Further description of the testing process
can be found in the [testing docs](../docs/testing.md)
